# Rock Paper Scissors

A command line Rock Paper Scissors game built in Python.

## Features
- Configurable number of rounds
- Proper win condition logic
- Input validation and error handling
- Play again option
- Multiplayer mode
- AI opponent tracks your move frequency and predicts your next move

## How to Run
python rock_paper_scissors.py

## Concepts Used
- Object Oriented Programming
- Error handling
- Defensive programming
- Input sanitization
- Frequency based AI opponent

- ## Note
This project is intended for personal/fun use. 
Contains humorous responses to repeated invalid inputs.
